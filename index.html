Capstone Optimization

TitanQ Solver Evaluation and Optimization Benchmarking

This capstone project focused on evaluating the usability and performance of the TitanQ quantum-inspired solver developed by InfinityQ. The project involved modeling optimization problems, benchmarking solution quality and runtime, and comparing TitanQ with CPLEX across scheduling and vehicle routing problem settings.

My role: Contributed to modeling, experimentation, result interpretation, and communication of findings across project iterations.

Key skills: Optimization modeling, benchmarking, data analysis, experimentation, technical communication.

Reflection: This project strengthened my ability to evaluate technical tools beyond surface-level performance claims. I learned how to compare methods systematically, interpret trade-offs, and communicate results in a way that balances technical rigor with practical relevance.

Web App FinTech Frontend

Hconnect – Credit Card Optimization Web Application

Hconnect is a web-based financial application designed to help users optimize credit card usage and maximize rewards based on spending behavior. The platform aims to make complex financial choices more understandable through a practical, user-friendly digital experience.

My role: Primarily responsible for frontend development, including responsive interface implementation, UI component development, and integration with backend APIs.

Key skills: React, HTML, CSS, JavaScript, API integration, component-based UI design, Git/GitHub collaboration.

Reflection: This project deepened my understanding of how frontend development supports product usability. A key challenge was integrating interface logic with backend data while maintaining clarity for users. Through this experience, I improved my ability to translate product requirements into practical and accessible web interfaces.

Machine Learning Finance

Portfolio Optimization with Transformer-Based Deep Learning

This project explored the use of transformer-based architectures within a deep reinforcement learning framework for portfolio management. The objective was to compare data-driven investment strategies with more traditional financial decision approaches.

My role: Worked on model understanding, experiment setup, evaluation framing, and interpretation of financial performance outcomes.

Key skills: Python, machine learning, deep learning concepts, financial modeling, performance evaluation.

Reflection: I gained stronger intuition about the gap between theoretical model performance and real-world financial decision-making. The project also helped me become more comfortable discussing technical model design in business-relevant terms.
